# YouBlog.Mvc
A Little Blog Build Whit ASP.NET MVC + EF6

Demo：http://youblog.yangshunjie.com/

###特点：
0. 文章使用Markdown编辑器；
0. 优化路由配置，文章、栏目、菜单等Url可以使用可读性强的英文（自动翻译）替代；
1. 文章可设置为独立页面；
2. 自定义菜单；
3. 使用.html伪静态；
